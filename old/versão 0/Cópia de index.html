O que você precisa preparar?

Para agilizar seu processo, tenha em mãos os seguintes documentos base para o ano-calendário 2025:

Documentos Pessoais

CPF, RG, Dados bancários (agência e conta), Comprovante de endereço, Título de eleitor, Certidão de nascimento/casamento/divórcio.

Comprovantes de Despesas

Recibos de médicos, dentistas, laboratórios, mensalidades de escolas, faculdades para você e dependentes.

Documentos Bancários, de Imóveis e de Veículos

Escrituras, matrículas, registros atualizados, CRLV, extratos bancários para fins de IR, extratos de aplicações financeiras.

Informes de Rendimento

Salário, pensão, pró-labore, aposentadoria ou aluguéis.
